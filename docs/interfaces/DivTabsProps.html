items

items: Type<NonEmptyArray<IDivTabsItem>, string>

Tabs. Transition between tabs can be implemented using:div-action://set_current_item?id=&item= — set the current tab with an ordinal number item inside an element, with the specified id;div-action://set_next_item?id=[&overflow={clamp\|ring}] — go to the next tab inside an element, with the specified id;div-action://set_previous_item?id=[&overflow={clamp\|ring}] — go to the previous tab inside an element, with the specified id.

The optional overflow parameter is used to set navigation when the first or last element is reached:clamp — transition will stop at the border element;ring — go to the beginning or end, depending on the current element.

By default, clamp.
