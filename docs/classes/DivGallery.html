Optionalitems

items?: Type<NonEmptyArray<Div>, string>

Gallery elements. Scrolling to elements can be implemented using:div-action://set_current_item?id=&item= — scrolling to the element with an ordinal number item inside an element, with the specified id;div-action://set_next_item?id=[&overflow={clamp\|ring}] — scrolling to the next element inside an element, with the specified id;div-action://set_previous_item?id=[&overflow={clamp\|ring}] — scrolling to the previous element inside an element, with the specified id.

The optional overflow parameter is used to set navigation when the first or last element is reached:clamp — transition will stop at the border element;ring — go to the beginning or end, depending on the current element.

By default, clamp.
