interpolator?: Type<(DivExpression | DivAnimationInterpolator), string>

Animation speed adjustment. When the value is set to spring, it’s a damped oscillation animation truncated to 0.7, with the damping=1 parameter. Other values correspond to the Bezier curve:linear — cubic-bezier(0, 0, 1, 1)ease — cubic-bezier(0.25, 0.1, 0.25, 1)ease_in — cubic-bezier(0.42, 0, 1, 1)ease_out — cubic-bezier(0, 0, 0.58, 1)ease_in_out — cubic-bezier(0.42, 0, 0.58, 1)
